## Project Fishing Roller
Software to make Rapala Pro Bass Fishing Rod for PS3 run on desktop computers(currently only going to target windows)

## Requirements
Need to have ViGEM Bus Driver so I can actually map to something
Recommend something like AntiMicroX to go from controller to keyboard and mouse(didn't make it map to keyboard and mouse directly cause I think game controller is more general and useful)

## Why? This is a piece of like 15+ year old tech that almost nobody has...
I wanted to play minecraft with a fishing rod :thumbsup:. Imagine the "aura loss" as the cool kids call it of being beaten in a duel with a fishingrod.
